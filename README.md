# recoil-folders

Проект собран с помощью: `CRA (craete react app)`

Цель: Написать приложение для наглядности работы `recoil`

Инструменты: `react` / `ts` / `recoil`

Как развернуть проект:
ветка `main` => `git clone` => `npm i` => `npm start`
